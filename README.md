# 📊 Customer Churn Analysis & Prediction Dashboard

## 📌 Project Overview
This project analyzes customer churn using the IBM Telco Customer Churn dataset. An interactive Power BI dashboard was developed to identify churn patterns, monitor key performance indicators (KPIs), and provide actionable business insights to improve customer retention.

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📂 Dataset
- IBM Telco Customer Churn Dataset
- Total Records: **7,032**
- Total Features: **21**

---

## 📈 Dashboard Features
- KPI Cards (Total Customers, Churned Customers, Churn Rate)
- Customer Demographics Analysis
- Customer Tenure Distribution
- Churn by Contract Type
- Churn by Payment Method
- Churn by Internet Service
- Interactive Slicers
- Business Insights Panel

---

## 💡 Key Business Insights
- Customers with **Month-to-month contracts** have the highest churn.
- **Electronic Check** users churn more frequently.
- **Fiber Optic** customers show higher churn than DSL users.
- Customers with longer contracts are more likely to stay.
